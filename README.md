# mjWordArt
An easy to use and colorful set of word arts made in CSS3.
- 15 style variations
- Easy to implement, you just need to add a couple of classes
- No images required

## How To Install
1. Extract the downloaded zip file (mjWordArt.zip)
2. Insert below codes to the <head> section of your page
`<link rel="stylesheet" type="text/css" href="mjWordArt.css" />`
